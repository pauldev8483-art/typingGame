# Typing game

## Key idea:
Build a lighweight brower based typing game that focuses on accuracy to build speed.

Most typing games/programs seem to focusing on gamification of typing focus on speed.
I want to improve my typing and would like to make sure that accuracy is the main focus, as its the mistakes that slow me down.


## Key features

Typing input and sentence to copy

User feedback and highlighting , I was thinking that words give scores, and muliple words correct give combos.
So to get a good score, accuracy is needed.

The program should save the scores, and have graph to show improvement. (I think It should save the errors, so It can create a map of the most errored keys)

use free texts, and random words, but also code. (maybe integrate AI generated content)

Have some minimal graphics for fun, with animations like fading stars, shaking text, to give user feedback and encouragement.

Basic sounds for feedback

Clean modern, dark UI, with nice clear font

I like typing.com, with clear indication of correct and incorrect letter, and what letter you are currently on.


## Language 
JS, but not sure on framework that is most suitable. I would like to be able to build and run it localy at work without node if possible. (the development version can have node, but could it be run and deployed localy - live server?)

Possibly Django if not, as I can create venv, and then could use sql database to store the results?
